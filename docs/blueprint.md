# **App Name**: TypeSprint

## Core Features:

- Dynamic Text Generation: Generate varied and challenging typing passages using an AI tool to ensure a fresh experience each time.
- Interactive Typing Interface: A clean input area where users type the displayed text, providing immediate visual feedback for correct and incorrect characters.
- Real-time Performance Metrics: Display live updates of Words Per Minute (WPM) and accuracy as the user types.
- Countdown Timer: Implement a clear, customizable timer that starts upon the first keystroke and signals the end of the test.
- Comprehensive Result Summary: Present a detailed summary at the test's conclusion, including final WPM, accuracy percentage, and a breakdown of errors.

## Style Guidelines:

- Color scheme: Light mode, promoting readability and focus on the task. Concepts: Precision, Focus, Efficiency.
- Primary color: A deep, professional blue-violet (#333399). Hue is inspired by concepts of focus and stability in technical environments.
- Background color: A very light, desaturated blue-violet (#EEEEF7), visually linked to the primary color while maintaining a clean, open canvas.
- Accent color: A vibrant cyan-blue (#3BAADBE), chosen to highlight interactive elements and provide a sense of progression and energy.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, perfectly suited for presenting text and metrics clearly.
- Use minimalist, outline-style icons for navigation and actions, ensuring they complement the clean interface without distracting from the main content.
- A centered, uncluttered layout that keeps the typing text, input area, and performance metrics prominently in the user's field of vision.
- Subtle, smooth animations for progress indicators and typing feedback (e.g., cursor movement, character highlight on error) to enhance user experience without being intrusive.
